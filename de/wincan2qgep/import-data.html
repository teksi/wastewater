Datenimport

Once the matching achieved, data can be imported using the Import button. A check will be first performed and the process will be stopped if any issue is found. See Problemlösungen.

Once every selected channels and observations are correctly filled, data is imported within TWW data model in the VSA-DSS KEK extension.

Bemerkung

Achtung: Importierte Daten können im Moment nicht einfach wieder gelöscht werden, es gibt kein „Wiederrufen/Zurück/Undo“. Wenn die Daten noch einmal importiert werden, werden sie noch einmal eingefügt und sind dann doppelt vorhanden.
