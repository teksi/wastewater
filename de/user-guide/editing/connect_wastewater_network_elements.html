3.3.4. Abwassernetzwerkselemente verbinden

Eine Anleitung zum Verbindung von Abwassernetzelementen in TWW

3.3.4.1. Allgemeines

  • TWW verfügt über ein Werkzeug um Abwassernetzelemente zu verbinden

3.3.4.2. Ein Haltung mit einem Knoten oder einer anderen Haltung verbinden

  • Es spielt keine Rolle, welcher Layer ausgewählt ist

  • Der Layer vw_tww_reach muss im Bearbeitungsmodus sein

  • Klicke den Connect wastewater network elements Knopf

  • If you move the mouse over a reach, that can be connected, it is marked (orange color): you can click

  • Den Mausezeiger zu einem Abwassernetzelement führen, mit dem das erste Objekt verknüpft werden soll (wieder werden wählbare Objekte markiert) und dort wieder klicken

Bemerkung

the topology-layers are turned on in the screenshot to show the effect of the connection.

  • Ein Fenster erscheint, in dem gewählt werden kann, welche Verknüpfung definiert wird. Wenn eine Haltung verknüpft wird, sieht das Fenster so aus:

Bemerkung

there is no options ‚Reach Point From‘ in this example, because in a connection from reach to reach, it is not allowed to connect a From-reach point to another reach.

  • Wenn die Verknüpfung gespeichert wurde, erscheint ein blauer Mitteilungsbalken oben im Kartenfenster

  • To show the new connection, save your edits, click on the TWW-SQL-Button (refresh network topology) and refresh the screen. Now you can see the new connection in the topology-layers

3.3.4.4. Eine Verknüpfung löschen

Zur Zeit gibt es kein spezielles Werkzeug, um eine Verknüpfung zwischen Abwassernetzwerkelementen zu löschen.

  • Wähle den Layer vw_tww_reach und aktiviere den Bearbeitungsmodus

  • Wähle das Info Werkzeug und doppelklicke die Haltung, welche geändert werden soll

  • Lösche den Wert in Feld rp_…_fk_wastewater_networkelement, um die Verknüpfung zu löschen

Bemerkung

Beachte die ‚VSA Erfassungsgrundsätze‘: Ein von-Haltungspunkt (rp_from_fk_wastewater_networkelement) soll nie mit einer anderen Haltung verknüpft werden.

For example, when you have a reach discharging directly into another one without a structure between the two, the upstream reach should be connected to the downstream reach (rp_from_fk_wastewater_networkelement = downstream reach). But inversely, the downstream reach shouldn’t be connected to the upstream one. This rule is for swwf-channels (secondary network). In pwwf-network, there must be always a node between two reaches.
