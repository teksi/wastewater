Digitalisieren detaillierter Geometrie für Entwässerungsrinnen

Allgemeines

Mit dem Zeichnen einer einfachen Linie kann eine Fläche zu einem Abwasserbauwerk hinzugefügt werden. Die Funktion erzeugt aus der Linie ein Rechteck mit 20cm Breite. Das ist speziell nützlich bei Entwässerungsrinnen (Klasse Normschacht).

Digitalisierungswerkzeug

  • Wählen Sie den vw_tww_wastewater_structure Layer

  • Benutzen sie das Objekte abfragen Werkzeug auf dem Objekt, welchem Sie eine Entwässerungsrinnen-Geometrie hinzufügen wollen

  • Im Objekt Attribute Fenster: gehe zum Aktionen-Menu und wähle Digitize Drainage Channel wählen. (Digitalisiere Entwässerungsrinne)

oder

  • Rechtsklick auf das Objekt und wählen Sie den vw_tww_wastewater_structure Layer und den Datensatz

  • Wählen Sie Digitize Drainage Channel

  • Jetzt mit dem Digitalisieren beginnen: einfach eine Linie mit 2 Klicks definieren.

  • Ein Rechteck mit 20cm Breite wird in der Klasse Abwasserbauwerk (Layergruppe Abwasserbauwerk) hinzugefügt

  • Um die Breite zu ändern muss Ctrl während dem Digitalisieren gedrückt werden. Nach dem zweiten Klick, kann die Breite im Popup-Fenster definiert werden.
