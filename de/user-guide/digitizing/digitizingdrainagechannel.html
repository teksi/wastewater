3.2.4. Digitalisieren detaillierter Geometrie für Entwässerungsrinnen

3.2.4.1. Allgemeines

Mit dem Zeichnen einer einfachen Linie kann eine Fläche zu einem Abwasserbauwerk hinzugefügt werden. Die Funktion erzeugt aus der Linie ein Rechteck mit 20cm Breite. Das ist speziell nützlich bei Entwässerungsrinnen (Klasse Normschacht).

3.2.4.2. Digitalisierungswerkzeug

  • Select the vw_tww_wastewater_structure layer

  • Benutzen sie das Objekte abfragen Werkzeug auf dem Objekt, welchem Sie eine Entwässerungsrinnen-Geometrie hinzufügen wollen

  • Right click the object and select the vw_tww_wastewater_structure layer

  • Wählen Sie Digitize Drainage Channel

  • Jetzt mit dem Digitalisieren beginnen: einfach eine Linie mit 2 Klicks definieren.

  • Die Fläche wird im Layer od_wastewater_structure erstellt.

An other way is to open the vw_tww_wastewater_structure form with the info tool and then choose Digitize Drainage Channel in the Actions menu of the form.
