3.2.6. Objekte löschen

Sie können Punkt- oder Linienobjekte (mit allen verknüpften Objekten) mit den vw_tww_(Abwasserbauwerk/Haltung) Layern löschen. Sie können auch nur ein spezifisches Objekt auswählen und löschen (z.B. einen Deckel).

3.2.6.1. Haltung löschen

  • Wählen Sie vw_tww_Haltung und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie die Haltung(en), die Sie löschen möchten, z.B. mit Einzelklick oder mit einem Auswahlwerkzeug.

  • Benutzen Sie das Menü Bearbeiten –> Ausgewähltes löschen zum löschen.

Bemerkung

Am einfachsten ist es, die Delete Taste auf Ihrer Tastatur zu drücken. Oder Sie können den Ausgewähltes löschen Knopf anklicken, welcher sich in der Digitalisierungswerkzeugleiste befindet.

Abhängig von ihren QGIS– Einstellungen kann die Digitalisierungswerkzeugleiste versteckt sein oder sich an einem anderen Ort befinden.

  • Deaktivieren Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Layern. Alle Änderungen werden in der Datenbank gespeichert.

Bemerkung

Das Löschen von Haltung entfernt ebenfalls alle verbundenen Bauwerkteile (z.B. Trockenwetterfallrohre). Der verknüpfte Kanal wird nur gelöscht, falls kein weiteres Abwassernetzelement verbunden ist. Wenn zwei Haltungen mit einen Kanal verknüpft sind, wird der Kanal nur gelöscht, wenn beide Haltungen entfernt wurden.

3.2.6.2. Normschacht oder anderes Abwasserbauwerk löschen

  • Wählen Sie vw_tww_Abwasserbauwerk und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie das Schachtobjekt aus, welches Sie löschen möchten

  • Zum löschen benutzen Sie Bearbeiten –>**Ausgewähltes löschen** .

  • Beenden Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Layern. Alle Änderungen werden in der Datenbank gespeichert.

Bemerkung

Das Löschen von Schachtbauwerken entfernt ebenfalls alle verbundenen Bauwerkteile (z.B. Deckel, Einstiegshilfen etc.) und alle verknüpften Abwasserknoten. Fremdschlüssel FS_Abwassernetzelement in Haltungspunkten, die mit einem solchen Abwasserknoten verknüpft waren, werden NULL gesetzt.

3.2.6.3. Lösche ein Abwasserbauwerk, aber behalte den Abwasserknoten und die Verknüpfungen auf diesen Abwasserknoten

Zuerst muss die Verknüpfung zwischen Abwasserbauwerk und Abwasserknoten gelöscht werden. Weil das Feld FS_Abwasserbauwerk im Abwasserknoten Objekt Attribut Fenster nicht sichtbar ist (dieses Feld würde in anderen Fällen, wo dieses Formular ebenfalls eingesetzt wird, keinen Sinn machen), muss folgendes Vorgehen gewählt werden:

  • Wählen Sie vw_tww_Abwasserbauwerk und wechseln Sie zum Bearbeitungsmodus.

  • Benutze das Werkzeug Objekte abfragen, um das Formular des zu löschenden Schachtes zu öffnen

  • Wechseln zum Reiter „Abwasserknoten“

  • Wähle den(die) Abwasserknoten, welche nicht gelöscht werden sollen

  • Klicke den Knopf Gewählte(s) Kindobjekt(e) lösen

  • OK klicken um das Formular zu schliessen.

Der Abwasserknoten ist jetzt nicht mehr Teil des Abwasserbauwerks

  • Wählen Sie das Schachtobjekt aus, welches Sie löschen möchten

  • Zum löschen benutzen Sie Bearbeiten –>**Ausgewähltes löschen** .

  • Beenden Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Layern. Alle Änderungen werden in der Datenbank gespeichert.

3.2.6.4. Deckel löschen

  • Wählen Sie vw_tww_Abwasserbauwerk und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie das Schachtobjekt aus, von welchem Sie den/einen Deckel löschen möchten.

  • Benutzen Sie das Werkzeug Objekte abfragen, um das Formular zu öffnen

  • Wechseln Sie zum Deckel -Reiter

  • Wählen Sie den(die) Deckel, welche Sie löschen möchten

  • Klicke den Knopf Gewählte(s) Kindobjekt(e) löschen um den/die Deckel zu löschen

  • Klicken Sie im Formular auf Speichern

  • Deaktivieren Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Layern. Alle Änderungen werden in der Datenbank gespeichert.

Zweiter Weg um einen Deckel zu löschen

  • Wähle den Layer vw_cover und aktiviere den Bearbeitungsmodus

  • Deckel auswählen

  • Benutzen Sie Bearbeiten –>**Ausgewähltes löschen** .

  • Den Editiermodus beenden

3.2.6.5. Bauwerksteile löschen

  • Wählen Sie vw_tww_Abwasserbauwerk und wechseln Sie zum Bearbeitungsmodus.

  • Wählen Sie das Schachtobjekt aus, von welchem Sie Bauwerksteile löschen möchten.

  • Benutzen Sie das Werkzeug Objekte abfragen, um das Formular zu öffnen

  • Wechseln Sie zum Reiter „Bauwerksteile“

  • Wählen Sie die Bauwerksteile, welche Sie löschen möchten

  • Klicken Sie auf den roten X Knopf um die Bauwerksteile zu löschen

  • Klicken Sie im Formular auf Speichern

  • Beenden Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Layern. Alle Änderungen werden in der Datenbank gespeichert.

3.2.6.6. Detailgeometrie löschen

Achtung

Wenn ein Objekt im Layer Abwasserbauwerk (Layergruppe Abwasserbauwerk) gelöscht wird, wird das ganze Abwasserbauwerk mit den zugehörigen Datensätzen in der Datenbank gelöscht und nicht nur die Detailgeometrie des Bauwerks!

  • Wähle Abwasserbauwerke (nicht vw_tww_Abwasserbauwerke) und wechsle in den Bearbeitungsmodus

  • Benutze das Teil löschen Tool aus der Erweiterte Digitalisierungswerkzeugleiste

  • Es erscheint der Kreuz-Cursor. Klicke auf das Teil, das gelöscht werden soll (es gibt keine Rückfrage ‚wollen sie wirklich…‘ mit einem Click ist der angeklickte Teil gelöscht!)

  • Beenden Sie den Bearbeitungsmodus und bestätigen Sie die Änderungen an den Layern. Alle Änderungen werden in der Datenbank gespeichert.
