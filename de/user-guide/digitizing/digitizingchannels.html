Kanäle Digitalisieren

Allgemeines

TWW hat einen Assistenten, um Kanäle korrekt zu erfassen und diese mit Schächten respektive Abwasserknoten oder Haltungen zu verbinden (und dabei die Abwasserknoten- und Haltungs-Topologie aufzubauen). Siehe Kapitel Der TWW Wizard (Zauberer).

Wählen Sie die Wizard Schaltfläche, drücken Start Data Entry und wählen Reach im Pulldown-Menü.

Achtung

Erfassen Sie Haltungen/Kanäle immer in Fliessrichtung des Kanales, indem sie beim von Abwasserknoten starten und beim nach Abwasserknoten aufhören.

Digitalisieren

  • Im Digitalisierungsmodus schnappt der Cursor automatisch auf das nächstgelegene Abwasserbauwerk oder Haltung zu. Mit einem Linksklick wird ein Linienanfang gezeichnet.

  • Mit beliebigen weiteren Klicks können Sie Zwischenpunkte des Haltungsverlaufs definieren. Sie können auch direkt einen anderen Kanalschacht anwählen, um einen geraden Kanal zu zeichnen.

  • Die Advanced Digitizing Werkzeuge können zusammen mit dem Wizard eingesetzt werden.

  • Mit einem Rechtsklick wird die Linienzeichnung beendet. Im Anschluss erscheint das vw_tww_reach Formular in dem Sie Daten im Reiter Allgemein eingeben können:

Bemerkung

Beachten Sie, dass der Endpunkt der Linie der Punkt ist, an welchem Sie zuletzt Links geklickt haben. Deshalb benötigen Sie, für die Digitalisierung einer einfachen Linie mit 2 Punkten, zwei Linksklicks um die Linie zu digitalisieren und einen Rechtsklick, um die Digitalisierung der Linie zu beenden.

  • Das Schnappen auf Abwasserknoten oder andere Haltungen kann in den Feldern HP_von/nach_FS_Abwassernetzelement kontrolliert werden.

Bemerkung

Wenn im Feld Bezeichnung (reach-identifier) im Reiter Allgemein keine Eingabe gemacht wird, übernimmt TWW die obj_id als Bezeichnung (kann später geändert werden). Als Vorgabewert ist die Bezeichnung der Haltung gleich dem ws_identifier des Kanals. Als Vorgabewert ist die Bezeichnung der Haltungspunkte gleich der obj_id der Haltungspunkte.

Bemerkung

Siehe Geometrieabgleichung für automatische Haltungspunkthöhen bei Schnappen auf Abwasserknoten mit Koten.

  • Für den Profiltyp erhalten Sie eine Liste von vordefinierten Profilen. Die Rohrprofile können in der pipe_profile Tabelle bearbeiten werden (Layergruppe Abwasserbauwerk).

  • Wenn Sie fertig sind klicken Sie den OK Knopf

  • Speichern Sie die Informationen des Layers, indem Sie auf **Stop Data Entry ** klicken.

Bemerkung

Bei Verwendung des Assistenten während der Erstellung von Haltungen verwenden die Vorgabefelder im Reiter Allgemein (und ausschliesslich diese Felder) den zuletzt eingegebene Wert. Die QGIS-Option Letzte Attributeingaben wiederverwenden* und die Vorgabewerte haben keinen Einfluss auf diese Felder.

  • Sie können einen Datensatz bearbeiten, indem Sie den Bearbeitungsmodus auswählen und dann mit dem Info Cursor auf das Objekt klicken, welches Sie bearbeiten möchten.

  • Wenn Sie den Bearbeitungsmodus nicht auswählen, können Sie nur die existierenden Daten anschauen.

  • Detaillierte Informationen zum Bearbeiten siehe das Kapitel Bearbeiten von bestehenden Daten.

Weitere Attribute und Klassen

Wenn eine Haltung digitalisiert wird, finden im Hintergrund eine Reihe von Schritten in der TWW Datenbank statt:

  • Ein neues Objekt wird in der Klasse Abwasserbauwerk („wastewater_structure“) hinzugefügt

  • Ein neues Objekt wird hinzugefügt und verknüpft in der Unterklasse Kanal (channel)

  • Ein neues Haltungsobjekt wird in der Tabelle Abwassernetzelement (wastewater_networkelement) und dessen Unterklasse Haltung (reach) generiert.

  • Zwei neue Haltungspunktobjekte werden hinzugefügt und verknüpft mit der Haltung (rp_from_node, rp_to_node)

Geometrieabgleichung

Beim Einfügen einer neuen Haltung mithilfe des Wizard bestimmen die Start- und Endpunkte der Liniengeometrie die Höhe der entsprechenden Haltungspunkte (siehe Abbildungen oben). Diese Werte können vor dem Speichern überschrieben werden. Beim Speichern werden die Höhenwerte HP_von_Kote und HP_nach_Kote verwendet, um den Z-Wert des ersten und letzten Eckpunkts einer Haltung zu definieren.

Beim Aktualisieren einer Haltung gibt es mehrere Fälle:

  1. Der erste/letzte Geometriepunkt der Haltung wurde nicht verändert, wohl aber HP_von_Kote / HP_nach_Kote. In diesem Fall wird der Wert HP_von_Kote / HP_nach_Kote verwendet, um den Z-Wert des ersten/letzten Punkts der Haltung festzulegen.

  2. Der erste/letzte Geometriepunkt der Haltung wurde geändert, die Werte für HP_von_Kote und HP_nach_Kote jedoch nicht. In diesem Fall legt die QGIS-Variable @tww_update_lvl_by_geom fest, ob die Werte für HP_von_Kote und HP_nach_Kote durch den Z-Wert des ersten/letzten Geometriepunkts der Haltung überschrieben werden.

  3. Sowohl der erste/letzte Geometriepunkt als auch HP_von_Kote / HP_nach_Kote wurden geändert. HP_von_Kote / HP_nach_Kote haben Vorrang.

  4. Weder der erste/letzte Geometrie-Vertex der Haltung noch HP_von_Kote / HP_nach_Kote wurden geändert. Es gibt keine Änderungen.

Synchronisation der Koten von Zwischenpunkten (Punkte zwischen den Haltungspunkten) funktioniert auch, wenn beim Digitalisieren auf 3D-Punkte auf einem anderen Layer geschnappt wird (z.B. ein Textfile als GPS-Messungen mit x,y,z Koordinaten, welches im QGIS-Projekt eingefügt wurde). Man hat dann einen 3D-Haltung, aber der Export zu INTERLIS kann diese Zwischenpunkt-Koten nicht exportieren, weil die VSA-DSS Version 2020 die 3D-Koordinaten für die Haltungsgeometrie nicht unterstützt.

Digitalisieren von Spülstutzen

Added in version 2025.0.

TWW ist so konfiguriert, dass Spülstutzen als Bauwerksteil von Kanälen erfasst werden.

Ein Spülstutzen kann erstellt werden

  • im Fenster „Feature-Attribute“ von vw_tww_Haltung mit dem Tab „Bauteile“. Fügen Sie ein Punkt-Kind_Objekt hinzu und digitalisieren Sie den Punkt auf der Karte. Oder

  • digitalisieren Sie einen neuen Punkt auf vw_Spülstutzen und wählen Sie den verbundenen Kanal für das Feld FS_Abwasserbauwerk auf der Karte aus.

Bemerkung

Eine Spülstutzen ist mit einem Abwasserbauwerk verbunden, die Geometrie ist jedoch völlig unabhängig von der Geometrie des Abwasserbauwerks. Wenn der Kanal verschoben wird, muss der Spülstutzen manuell mit verschoben werden.
