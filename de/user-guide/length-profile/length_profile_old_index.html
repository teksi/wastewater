Längsschnitte

Eine Anleitung wie man einen Abschnitt wählt und einen Längsschnitt zeigt

Allgemeines

TWW hat eine einzigartige Funktion um Längenprofile darzustellen. Wählen Sie den Längenprofil Knopf um zu starten.

Das Längenprofil Fenster öffnet sich.

Wählen Sie den vw_tww_Deckel Layer um einen Startpunkt auszuwählen.

Wählen Sie dann den nächsten Kanalschacht - Es ist nicht zwingend den nächstgelegenen zu wählen. Das Werkzeug wählt automatisch die dazwischenliegenden Kanalschächte. Sie können das mehrere Male wiederholen. Gibt es keinen Anschluss erhalten Sie eine Warnmeldung. Rechtsklick um Ihre Auswahl zu beenden.

Im Profil Fenster haben Sie jetzt Ihr Längenprofil. Sie können detaillierte Informationen über die Kanalschächte finden. Wenn Sie über dem Kanalschacht im Profil Fenster schweben, können Sie sehen dass die zugehörige Arbeitsfläche in grün aufleuchtet

Das selbe gilt für die Haltungen

Drucken

Wählen Sie den Print Knopf, um das Profil zu Ihrem ausgewählten Drucker zu senden. Wenn Sie einen PDF Drucker installiert haben, können Sie es als eine PDF Datei speichern
