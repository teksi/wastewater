TEKSI Module Management Tool

TMMT ersetzt das DataModel Tool, das in QGEP verfügbar war

Das neue Plugin enthält ein fortschrittliches Tool, das in der Lage ist, eine TWW-Datenbank zu initialisieren (und zukünftig auch zu aktualisieren).

Es unterstützt Systemadministratoren und Power-User und sollte nicht von Anwendern verwendet werden, da es bei unsachgemässer Anwendung zu Datenverlusten führen kann.

Für anspruchsvolle Fälle ist es auch möglich, die Datenbank manuell zu aktualisieren <../db-update-pum/index.html>“.

Fehlerbehebung

In case you encounter issues or errors when using TMMT, make sure to look at the logs, as it may contain useful information (including errors returned by underlying tools).

Make sure to include this information if submitting a bug report or `asking for support https://www.teksi.ch/modules/#prestataires`_.
