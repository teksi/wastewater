4.2.2. pgAdmin verwenden

4.2.2.1. SQL Abfrage

Bemerkung

  • Wenn Sie das markierte Symbol 1 anklicken öffnet dies das SQL Fenster. Das SQL Symbol ist nur aktiv, wenn Sie mit einer Datenbank verbunden sind.

  • Ein Klick auf 2 führt den eingetippten SQL Befehl aus.

4.2.2.2. Datenbank Speicherauszug

Bemerkung

  • Öffnen Sie pgAdmin4 und verbinden Sie sich mit der Datenbank

  • Rechtsklicken Sie das Schema, welches Sie sichern wollen (in der Baumstruktur unter „Schemas“)

  • Geben Sie den Orndernamen ein, unter welchem der Speicherauszug abgespeichert werden soll

  • Klicken Sie den Option-Tab und überprüfen Sie die folgenden Optionen

Bemerkung

  • Nachdem der Prozess abgeschlossen ist, zeigt das Fenster rechts unten, dass alles gut gegangen ist
