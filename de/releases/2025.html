2025.0.2

Die wichtigsten Merkmale

  • Erste Version von TEKSI Abwasser für QGIS 3.40 LTR und installiert über den PostgreSQL Update Manager (PUM) und das TEKSI Module Management Tool (TMMT). Siehe installation guide für mehr Informationen wie TEKSI Abwasser vis TMMT installiert wird.

Bemerkung

Check the new release video to see how these features support ‚Clean waters and an environment worth living in!‘

  • Neue Stile und Symbolisierungen

  • Demo-Daten Installation mit TMMT

installation guide für weitere Infos zur Installation von TEKSI Abwasser via TMMT.

Migration von 2024.X zu 2025.X

  • Da mit dieser Version eine neue Datenbankstruktur eingeführt wird, ist eine Migration von 2024.X auf 2025.X erforderlich.

    Eine einfache Möglichkeit zur Datenmigration besteht darin, die Daten aus der 2024.X-Datenbank im INTERLIS-Format zu exportieren.

Bemerkung

Bitte installieren Sie das TWW-Plugin 2025.X in einem separaten QGIS-Profil und verwenden Sie die Version 2025.X anstelle der letzten Version 2024.X für den Datenexport, um sicherzustellen, dass Sie den vollständigen VSA-DSS-Datensatz erhalten. Die Version 2024.X des Plugins weist fehlende Attribute auf, die mit Der DSS-Version 2020 eingeführt wurden (https://github.com/teksi/wastewater/issues/902). Ein Patch hierfür wurde erst in TWW Version 2025 hinzugefügt (https://github.com/teksi/wastewater/pull/568).

Bemerkung

Zukünftig kann dies direkt innerhalb der Datenbank erfolgen, wenn das TMMT-Plugin neben der Installation der Datenbank auch Aktualisierungen unterstützt.

and import it into a new empty 2025.X database (also use the latest 2025.X release for that).
