Extension AG-64/AG-96
==================

This represents a guide on effectively use the AG-64/AG-96 extension in TWW.

.. note::

 The extensions for AG-64/AG-96 support are not yet included in the official TWW release 2024.0. The extension can be loaded based on the plugin and datamodel versions of https://github.com/teksi/wastewater/pull/399
