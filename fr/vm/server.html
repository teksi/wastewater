serveur QGIS

You can download a VM with an older TWW setup that also has QGIS Server installed from http://docs.qtibia.ro/qgisplatform.vdi.xz .

The VM has working installations of QGIS Server and QGIS Web Client (QWC).

Accès aux services Web

The Apache server is setup to respond to requests pointing to the http://qgisplatform.demo. Accessing the above link with the Internet Browser of the VM will take you to the starting page of QWC depicted in the below image.

Si vous souhaitez accéder aux services Web en dehors de la MV, vous devez éditer les fichiers hosts sur votre machine et pointer sur qgisplatform.demo comme IP de la MV.

Note

Si vous avez configuré la MV avec l’adaptateur réseau en mode NAT, seule la MV hôte pourra accéder aux services. Si vous souhaitez que toutes les machines de votre réseau local puissent y accéder, vous devez configurer l’adaptateur réseau en mode pont.
