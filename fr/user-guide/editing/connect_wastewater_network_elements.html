3.3.4. Connexion des éléments du réseau assainissement

This represents a guide on how to connect wastewater network elements in TWW.

3.3.4.1. Généralités

  • TWW has a tool to connect wastewater network elements

3.3.4.2. Comment connecter les tronçons aux noeuds ou les noeuds à d’autres noeuds

  • Peu importe la couche sélectionnée

  • The vw_tww_reach layer has to be in edit mode

  • Cliquez sur le bouton Connecter les éléments du réseau assainissement

  • If you move the mouse over a reach, that can be connected, it is marked (orange color): you can click

  • Déplacez la souris sur l’élément du réseau auquel le premier objet doit être connecté (rappel: les objets sélectionnables sont surlignés) et cliquez à nouveau

Note

the topology-layers are turned on in the screenshot to show the effect of the connection.

  • Une fenêtre apparaît, dans laquelle cous pouvez choisir quelle connexion sera définie. Si vous connectez un tronçon, la fenêtre aura cet aspect:

Note

there is no options “Reach Point From” in this example, because in a connection from reach to reach, it is not allowed to connect a From-reach point to another reach.

  • Si tout est OK, vous verrez apparaître un message dans une barre bleue, au-dessus de la vue de la carte

  • To show the new connection, save your edits, click on the TWW-SQL-Button (refresh network topology) and refresh the screen. Now you can see the new connection in the topology-layers

3.3.4.4. Comment supprimer une connection

Pour l’instant, il n’y a pas d’outil spécial permettant de supprimer une connexion entre éléments du réseau.

  • Select the vw_tww_reach layer and set it to edit mode

  • Cliquez sur le bouton Identifier objets et sélectionnez le tronçon à modifier par double-clic

  • Supprimez la valeur dans le champ rp_…_fk_wastewater_networkelement pour supprimer la connexion

Note

Soyez conscient des principes de saisie selon le VSA : Un point de tronçon « depuis » (rp_from_fk_wastewater_networkelement) ne devrait jamais être connecté à un autre tronçon.

For example, when you have a reach discharging directly into another one without a structure between the two, the upstream reach should be connected to the downstream reach (rp_from_fk_wastewater_networkelement = downstream reach). But inversely, the downstream reach shouldn’t be connected to the upstream one. This rule is for swwf-channels (secondary network). In pwwf-network, there must be always a node between two reaches.
