2024.X

Here you can find information about the 2024.X release of TEKSI wastewater

Main features

  • First stable release of TEKSI wastewater module for QGIS

  • Support of SIA 405 Wastewater (pipe cadastre) and VSA-DSS 2020.1 (general wastewater management plan)

  • Support of VSA-KEK (TV inspections) 2020.1

  • INTERLIS import/export

All main available features are described in detail in the features guide. and in the user guide.

New functionalities will be tagged with the release version directly in the documentation as well as in the releases.

Installation

As this release was published before the development of the TEKSI Module Management Tool plugin (TMMT), it has to be installed manually directly from the assets available on the release page.
